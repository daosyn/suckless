# suckless

personal patches for [suckless tools](https://suckless.org), specifically `dwm` and `st`

## installation

clone repository

```shell
git clone --recurse-submodules https://github.com/daosyn/suckless.git
```

install both `dwm` and `st`

```shell
make all
```

clean up `dwm` and `st` directories

```shell
make clean
```
